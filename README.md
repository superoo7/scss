# SCSS compiler
This repo is for scss compile to css with the help of gulp.

## Prerequisite
* node.js
* npm / yarn
* git


## Installation
1. 	go to your desired folder.
2. `git clone git@github.com:superoo7/scss.git`
3. `cd scss`
4. `yarn install` or `npm install`
5. `gulp` then BrowserSync will do its job
